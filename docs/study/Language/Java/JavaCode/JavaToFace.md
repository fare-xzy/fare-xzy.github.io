* System.arraycopy(Object src,int srcPos,Object dest,int destPos,int length)实现数组之间的复制
    * 参数含义
        * src 原数组
        * srcPos 源数组要复制的起始位置
        * dest 目标数组
        * destPos 目的数组放置的起始位置 
        * length 复制的长度
    * PS：arraycopy()方法可以实现自身复制即自己复制自己
